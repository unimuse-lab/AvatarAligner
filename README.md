# Avatar Aligner for Unity

Unityエディタ上のVRChatアバターを、指定した間隔で正方向（+X）または負方向（-X）に等間隔整列させるためのツールです。
身長差の比較、衣装のバリエーション確認、集合写真のセットアップなどを効率化します。

## ⚠️重要：データの保存仕様について
本ツールは、VCC（VRChat Creator Companion）環境での利用を想定し、ツール本体を `Packages` フォルダに配置しても設定が保持されるよう設計されています。

* ツールの設定値（間隔、方向、基準など）は、プロジェクト内の `Assets/UniMuseData/AvatarAligner/` フォルダに `AvatarAlignerData.asset` として自動保存されます。
* プロジェクト間で設定を共有したい場合は、上記の `.asset` ファイルをコピーしてください。

## 主な機能
* **等間隔整列**: X軸方向に指定した数値（Interval）でアバターを並べます。
* **柔軟な並び替え順**: 
    * **HierarchyOrder**: ヒエラルキーの上から順に並べます。
    * **PositionOrder**: 現在のX座標が端にあるものから順に並べます。
* **選べる開始地点**:
    * **FirstObject**: 1体目のアバターの位置を動かさず、そこを起点にします。
    * **WorldZero**: ワールド座標の原点（X=0.0）を起点にします。
    * **CustomX**: 任意のX座標を指定して整列を開始できます。
* **VRChatアバター自動認識**: `VRC_AvatarDescriptor` コンポーネントを持つオブジェクトのみを抽出して処理します。
* **アンドゥ（Undo）対応**: 整列実行後、Unity標準の Undo (Ctrl+Z) で元の位置に戻せます。

## 使用方法
1. **ウィンドウを開く**: 上部メニュー `UniMuse.lab` > `Avatar Aligner` を選択。
2. **対象を選択する**:
    * ヒエラルキー上で整列させたいアバターを複数選択します。
    * **何も選択していない場合**: シーン内のすべてのVRChatアバターを対象にします。
3. **設定を調整する**:
    * **間隔 (Interval)**: アバター同士の距離を入力します。
    * **並べる方向**: 
        * **Positive**: 正方向 (+X) に並べます。
        * **Negative**: 負方向 (-X) に並べます。
4. **実行**: 「整列を実行」ボタンを押すと、アバターが整列されます。

## 注意点
* **対象の判定**: オブジェクトに `VRC_AvatarDescriptor` または `VRCAvatarDescriptor` コンポーネントがアタッチされている必要があります。
* **座標軸**: 本ツールは **X軸方向** の整列のみに対応しています。
* **非表示オブジェクト**: 非アクティブなオブジェクトも、アバターコンポーネントがあれば整列の対象に含まれます。
